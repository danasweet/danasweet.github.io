Catfinity

Catfinity is a simple but fun web application that generates a new cat image/gif with the click of a button. It's a Sinatra app that uses the Cat API to scrape Tumblr for cat images, and incorporates basic jQuery to populate a new image upon clicking the left or right arrow. It was a solo project built in an afternoon.
